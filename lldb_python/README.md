1. 先使用如下命令确定是否已经有`lldbinit`文件存在`cat ~/.lldbinit`
2. 如果没有`lldbinit`，则使用终端执行`touch ~/.lldbinit`或者直接从这里下载现成的[lldbinit][lldbinit]
3. 如果已经有`.lldbinit`那么将下载下来的`.lldbinit`加到原本`.lldbinit`的后面


			现在说明一下文件结构
			├── README.md
			├── douyu
			├── panda
			├── qmtv
			├── .lldbinit
			├── yy
			└── zhanqi
